# Data_projects_TripleTen
Projects I worked on while attending the Data Analysis TripleTen Bootcamp.


| Project number | Title | Description |
| :-----------: | ----------- |----------- |
| 1 | Manhattan vacation rental market analysis project| The project task was to analyze user data and find possible rentable properties for investment opportunities|
| 2 | E-commerce retention rate project |The project task was to Create a 3-stage conversion funnel and monthly retention cohort analysis in Excel using SQL-like filtering, pivot tables, VLOOKUP, and DATEDIF to measure user drop-off and repeat purchase behavior.|
| 3 | Profits and Losses Data Visualization| The project task was to study the superstore's operations and increase its profitablity to avoid bankruptcy |
| 4 | understanding return rates with tableau | The project's task was to build an interactive Tableau dashboard from Airbnb data, visualizing KPIs (listings, prices, occupancy, revenue), neighborhood/bedroom breakdowns via bars, a geographic map, and dynamic filters. |
